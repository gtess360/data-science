## Projects

### 1. Automobile Price Prediction
Predicting car prices using 25 independent variables to help 
management understand how design and specification choices affect pricing.

**Key highlights:**
- Dataset: 201 records, 25 features
- Best model: Random Forest (R² = 0.9462, MAE = $1,702)
- 6 models compared: Linear Regression, Ridge, Lasso, 
  Decision Tree, Random Forest, Gradient Boosting
- Top price drivers: engine size, curb weight, horsepower

**Tools used:** Python, Pandas, Scikit-learn, Matplotlib, Seaborn
